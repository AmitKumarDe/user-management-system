# User Management System API

A secure and scalable RESTful User Management API built with Node.js, Express.js, MongoDB, and Mongoose following the MVC architecture.

## Features

### Authentication & Authorization

* User Registration
* User Login
* JWT Access Token Authentication
* Refresh Token Implementation
* Cookie-Based Authentication
* Protected Routes
* Role-Based Access Control (Admin/User)

### User Management (CRUD)

* Create User
* Get All Users
* Get User By ID
* Update User
* Delete User (Admin Only)

### Advanced Features

* Pagination
* Search Functionality
* Sorting Support
* Request Validation using Zod
* Password Hashing with bcrypt
* MongoDB Query Optimization

### Security Features

* Password Hashing
* JWT Authentication
* Refresh Token Storage
* Protected Endpoints
* Role-Based Authorization
* Input Validation

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (jsonwebtoken)
* bcrypt
* Zod
* Cookie Parser

## API Highlights

* RESTful API Design
* MVC Architecture
* Middleware-Based Authentication
* Admin/User Role Management
* Search, Sort & Pagination Support
* Input Validation & Error Handling

## Project Structure

MVC (Model-View-Controller) architecture is used to maintain clean, scalable, and maintainable backend code.
